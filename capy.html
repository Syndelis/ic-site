Uma biblioteca para construção e simulação de Autômatos Celulares

Com uma simples interface e alta eficiência

Por Brenno Lemos e Alexandre B. Pigozzo , Março de 2020

Introdução

A ferramenta, criada como parte do projeto de Iniciação Científica da UFSJ, tem como objetivo simplificar o processo de modelagem por Autômato Celular.

A biblioteca é uma extensão para Python que introduz uma interface Orientada a Objetos para a criação e simulação dos modelos, utilizando matplotlib para a representação gráfica dos resultados.

Seu primeiro AC

from ca import CA, plot

class MeuCA(CA):
  def rule(self, x, y):
    return self[x, y] - 1 or 0

c = MeuCA(30)
plot(c, out="test.pdf")
